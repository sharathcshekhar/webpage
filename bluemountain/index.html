Overview

BlueMountain is a pluggable data management solution for modern mobile platforms like Android. The goal of BlueMountain is to allow data management mechanisms and policies to be implemented independently of core app logic. BlueMountain allows a user to install data management solutions as apps, install multiple such solutions on a single device, and choose a suitable solution each for one or more apps. It allows app developers to focus their effort on app logic and helps the developers of data management solutions to achieve wider deployability. It also gives increased control of data management to end users and allows them to use different solutions for different apps. BlueMoutain works on regular Android phones and does not require a custom ROM or root permissions. BlueMountain also achieves backward compatibility using Reptor byte-code instrumentation techniques that were earlier developed in RMS Lab.
